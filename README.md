# DCC-GARCH Project

## Overview
This repository contains datasets, scripts, and report files for a university coursework project on Dynamic Conditional Correlation (DCC-GARCH) modeling.

The project investigates time-varying correlations among:

- Bitcoin
- Gold
- S&P 500

## Objectives

- Estimate dynamic volatility
- Analyze safe-haven properties
- Examine inflation hedging performance
- Compare asset behavior during market stress

## Repository Structure

dataset/ : raw and processed data

code/ : Python 

figures/ : generated charts

report/ : final report PDF
